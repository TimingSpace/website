void *

Introduction

void * is special pointer in C++,which
1. has no type of course;
2 can point to any kind of variate such as int, float and complex type;
3 cannot be directly dereferenced since there is no type information to guide the compiler in interpreting the underlying bit pattern.

Usage

how to get the content where void* point to?

how to change the content where void* point to?

void* has no type, so we can not access the content directly, and to access its content, we should convert its type(void) to a certain type such as int, bool, string and so on. For example

void onMouse( int event, int x, int y, int, void* u)
        {
          bool * flag=(bool *)u;
          if(event==CV_EVENT_LBUTTONDOWN)
          {
            if(x>600&&x<620&&y>460&&y<470)
              {
                  *flag=!(*flag);
                  std::cout<<*flag<<std::endl;
              }
          }
        }
