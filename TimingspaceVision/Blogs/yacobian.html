Jacobian Matrix

Introduction

Jacobian Matrix is first-order direction derivative of Multi input and multi output function. Each row of Jacobian matrix is the gradiant of relative sub function.

For detail, please refer to a tutprail by UNSW

Please notice that there may be a small mistake in Page 13 line 2 where the Jacobian Matrix should not be mxn but nxm when f is a function from m variates to n variates.
